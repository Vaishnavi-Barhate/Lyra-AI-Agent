## Coversations Meet Calenders 

**Lyra AI** is an intelligent **AI receptionist** designed to automate appointment scheduling using **natural language conversations**.  
It is especially useful in **hospitals and service-based organizations**, where quick and accurate scheduling is critical.

Users can simply speak or type natural phrases, and Lyra will schedule appointments directly in **Google Calendar**, eliminating manual booking.

> “Schedule a patient appointment tomorrow at 3 PM”  
> “Book a follow-up next Monday at 10 AM”  
> “Doctor consultation in two hours”  

---

## Requirements
- Python 3.9+ and FastAPI
- Google Cloud project with Calendar API enabled
- OAuth credentials for Google Calendar


### Python Libraries
- `dateparser`
- `google-auth`
- `google-api-python-client`
- `requests`
